Cara Menjalankan Aplikasi
1. Menjalankan Database
  Jalankan XAMPP.
  Aktifkan Apache dan MySQL.
  Buka phpMyAdmin melalui browser.
  Import database db_optik_231011403084.
  Pastikan tabel login_user, kacamata, dan resep sudah tersedia.
2. Membuka Project
  Buka NetBeans IDE.
  Pilih menu File → Open Project.
  Pilih folder project Aplikasi Optik.
  Pastikan library MySQL Connector/J sudah terpasang.
3. Menjalankan Program
  Klik kanan nama project.
  Pilih Run Project.
  Form Login akan muncul.
4. Login ke Sistem
  Username: meysya
  Password: 123
5. Menggunakan Aplikasi
  Setelah login berhasil, pengguna dapat:
  Mengelola data kacamata
  Mengelola data resep
  Melakukan pencarian data
  Mencetak laporan
  Logout dari sistem
